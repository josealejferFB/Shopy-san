# Plantilla inicial - Shopy-San

Se añaden tres pantallas básicas: `Login`, `Register`, `Home` y la configuración mínima en `App.js` para usar navegación.

Dependencias recomendadas (instalar antes de ejecutar):

1. Instalar dependencias de React Navigation (con Expo):

```bash
npx expo install react-native-screens react-native-safe-area-context
npm install @react-navigation/native @react-navigation/native-stack
```

2. Ejecutar la app con Expo:

```bash
npm install
npm run start
``` 

Notas:
- Las pantallas incluyen componentes básicos y navegación. Sustituye la lógica de login/registro por tus llamadas a la API.
- Si usas Yarn, adapta los comandos.
